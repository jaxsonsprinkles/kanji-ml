# Hiragana Recognizer

Learn Japanese hiragana through handwriting recognition.

## What It Does
- Draw hiragana characters with your mouse
- Shows top 3 guesses with confidence scores

## Tech Stack
- Python + TensorFlow/Keras
- Tkinter (GUI)
- Custom dataset (544 handwritten samples)

## How to Run
```
pip install tensorflow pillow numpy
python gui.py
```

## Why I Built This
Learning Japanese and wanted a tool to practice hiragana recognition with my own handwriting style.

Built in 1 week
